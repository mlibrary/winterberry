Import Resources and Associated Metadata to Monograph

The resources can be imported within the monograph by the following steps:

  1. Log into host bulleit-1.

  2. cd /hydra/heliotrope-production/current

  3. sudo -u heliotrope-production RAILS_ENV=production bundle exec script/import -m <monograph_noid> -d /mnt/umptmm/<publisher>/<ebook_isbn>_<author_last_name>/resources/<ebook_isbn>_<author_last_name>

monograph_node
Monograph NOID
publisher
Publisher name, e.g. UMP for University of Michigan Press or MPS for University of Michigan Publishing Services.
ebook_isbn
Ebook ISBN
author_last_name
Author last name
