Monograph Processing

Monograph processing steps include the following:

  1. Merge EPUB Resource Caption and Alternative Text with Resource Metadata.

  2. Import Resources and Associated Metadata to Monograph.

  3. Process EPUB Resource References.

The steps above invoke project scripts that require the directory path for the specific publisher monographs that exist on the University of Michigan Library Fulcrum publishing platform. Currently, the publisher monograph directories reside on the shared drive:

FULCRUM_DRIVE = //Sharedgroups/Information Management/Fulcrum

Below is a list of relative directories that contain monograph files for specific publisher sites:

  • University of Michigan Press - ${FULCRUM_DRIVE}/UMP

  • University of Michigan Publishing Services - ${FULCRUM_DRIVE}/MPS

The publisher site directory can be specified to the project scripts by either:

  • using the -p command line option, or

  • setting the current working directory to be the publisher site directory.
